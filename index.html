💡 ВЫВОДЫ ПО СЦЕНАРИЯМ
🔴 Пессимистичный

Убыточный сценарий. Не достигается безубыточность. Требуется увеличение трафика или конверсии минимум в 3 раза.

🟡 Базовый

Реалистичный сценарий. Окупаемость 6 месяцев. Стабильная прибыльность при средних показателях рынка.

🟢 Оптимистичный

Высокорентабельный сценарий. Быстрая окупаемость менее 1 месяца. Возможен при премиум-локации и сильном бренде.

📅 ПОМЕСЯЧНАЯ ДИНАМИКА ДЕНЕЖНОГО ПОТОКА
1️⃣ ИСХОДНЫЕ ДАННЫЕ

Введите свои параметры во вкладке "Исходные данные" для каждого из трех сценариев:

  • Пессимистичный: низкий трафик, минимальная конверсия
  • Базовый: реалистичные средние показатели
  • Оптимистичный: высокий трафик, максимальная конверсия
2️⃣ ОСНОВНЫЕ ПОКАЗАТЕЛИ

🚶 Трафик - Количество посетителей магазина в день

🎯 Конверсия - % посетителей, совершивших покупку (15-35% для розницы)

💰 Средний чек - Средняя сумма одной покупки в рублях

💎 Валовая маржа - % наценки от выручки после себестоимости товара

3️⃣ РАСХОДЫ

💸 OPEX - Операционные расходы: ежемесячные затраты на содержание

🏗️ CAPEX - Капитальные затраты: единоразовые инвестиции при открытии

4️⃣ КЛЮЧЕВЫЕ МЕТРИКИ

⭐ EBITDA - Прибыль до вычета процентов, налогов и амортизации

🎯 Точка безубыточности - Минимальная выручка для покрытия расходов

⏱️ Срок окупаемости - Время возврата первоначальных инвестиций

📊 ROI - Рентабельность инвестиций в процентах

5️⃣ КОЭФФИЦИЕНТ СЕЗОННОСТИ

📅 Сезонность - Корректирующий коэффициент спроса по месяцам (1.00 = норма)

• Коэффициент < 1.00 означает снижение спроса (например, 0.73 = -27% от нормы)

• Коэффициент > 1.00 означает повышение спроса (например, 1.12 = +12% к норме)

• Настройте коэффициенты под специфику вашего бизнеса

6️⃣ АНАЛИЗ РЕЗУЛЬТАТОВ

• Смотрите вкладку "Сводка" для сравнения сценариев

• Вкладка "Динамика" показывает накопление денежного потока с учетом сезонности

• При отрицательной EBITDA проект не окупается - корректируйте параметры

📚 ИСТОЧНИКИ ДАННЫХ

• Средняя конверсия в рознице: 20-40% (Retail TouchPoints, 2024)

• Типичные CAPEX для розницы: 1-3 млн руб (отраслевая статистика 2026)

• Оптимальная EBITDA маржа: 10-20% для устойчивого бизнеса
